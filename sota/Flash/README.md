# ProvPlug: Enhancing Flash with LLM-Driven Plugins

This directory contains the implementation of ProvPlug, a novel framework that enhances the Flash model with our LLM-driven plugins.

> **Flash: A Comprehensive Approach to Intrusion Detection via Provenance Graph Representation Learning**  
> Mati Ur Rehman, Hadi Ahmadi, Wajih Ul Hassan  
> *2024 IEEE Symposium on Security and Privacy (SP)*  

We provide the code and execution results for the Flash model augmented with ProvPlug plugins, as reflected in `results`.

To see the detailed results, go to the bottom of each .ipynb file.
